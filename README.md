# Portfolio

Static portfolio website for Abhishek Kumar.

Deployment
- A GitHub Actions workflow is included at `.github/workflows/pages.yml` which automatically deploys the repository to GitHub Pages whenever you push to the `main` branch.
- No build step is required — the workflow uploads the repository contents and deploys them as Pages.

Notes
- If you want a custom domain, add a `CNAME` file with your domain to the repository root.
- To change the published branch or configure Pages manually, visit the repository Settings → Pages.
